# Instalación

## Agregar al proyecto

```bash
go get github.com/go-pdf/fpdf
```

Para actualizar a la última versión:

```bash
go get -u github.com/go-pdf/fpdf
```

## Importar en el código

```go
import "github.com/go-pdf/fpdf"
```

## Requisitos

- Go 1.13 o superior (por el sistema de módulos)
- **No tiene dependencias externas.** Usa solo la librería estándar de Go.
- Compatible con Linux, macOS y Windows.
