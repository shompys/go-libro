# mark3labs/mcp-go

Implementación en Go del Model Context Protocol (MCP), un estándar abierto que permite conexiones seguras y controladas entre aplicaciones de IA y fuentes de datos/herramientas externas.

```bash
go get github.com/mark3labs/mcp-go
```
